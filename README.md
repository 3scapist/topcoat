# TopCoat Mobile

An experimental CSS microlibrary for building mobile apps.

## Embedded Webview Targets

- iOS 5+
- Android 2.2+
- BlackBerry 6+
- Windows Phone 8+

## Mobile Browser Targets

- Firefox for Android
- Opera Mini
- Android 2.2+ system browser
- Mobile Safari 5+
